#### (Q) Add an external style sheet with the URL: `"mystyle.css"`.

<h5>Solution</h5>

###### Code

```HTML

<head>
    <link rel="stylesheet" href="mystyle.css">
</head>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
</body>

```
