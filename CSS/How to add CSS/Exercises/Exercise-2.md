#### (Q) Set the page's background color to red.

<h5>Solution</h5>

###### Code

```HTML

<style>
    body {
        background-color: red;
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
</body>

```
