#### (Q) Set the text color to red, for elements with `class="colortext"`.

<h5>Solution</h5>

###### HTML

```HTML

<style>
    .colortext {
        color: red;
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p class="colortext">This is a paragraph</p>
    <p class="colortext">This is a paragraph</p>
</body>

```
