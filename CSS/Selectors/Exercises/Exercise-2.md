#### (Q) Set the text color to red, for th element with `id="para1"`.

<h5>Solution</h5>

###### Code

```HTML

<style>
    #para1 {        
        color: red;
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p id="para1">This is a paragraph</p>
</body>

```
