#### (Q) Set the background color of the `<h1>` element to `"lightblue"`.

<h5>Solution</h5>

###### Code

```HTML

<style>
    h1 {
        background-color: lightblue;
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
</body>

```
