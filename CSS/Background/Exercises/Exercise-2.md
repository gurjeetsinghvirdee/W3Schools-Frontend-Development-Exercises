#### (Q) Set `"paper.gif"` as the background image of the page.

<h5>Solution</h5>

###### Code

```HTML

<style>
    body {
        background-image: url(/Assets/paper.gif);
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
</body>

```
