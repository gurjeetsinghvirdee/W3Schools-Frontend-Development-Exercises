#### (Q) Make the background image repeat only vertically.

<h5>Solution</h5>

###### Code

```HTML

<style>
    body {
        background-image: url(/Assets/background-image.jpg);
        background-repeat: repeat-y;
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
</body>

```
