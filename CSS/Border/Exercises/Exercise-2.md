#### (Q) Use the correct border property to set the border color to `"red"`.

<h5>Solution</h5>

###### Code

```HTML

<style>
    p {
        border-style: dotted;
        border-width: 4px;
        border-color: red;
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
</body>

```
