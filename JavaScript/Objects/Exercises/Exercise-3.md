#### Create an **object** called `person` with `name = John, age = 50`. Then, access the **object** to  `alert("John is 50")`.

<h5>Solution</h5>

###### Code

```JavaScript

const person = {
    name: "John",
    age: 50
};

alert(person.name + " is " + person.age);

```
